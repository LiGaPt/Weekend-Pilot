     - inherited stdio keeping the process open

   - Add `frontend/scripts/fixtures/playwright-web-server-child.mjs`:

     - spawn the wrapper around the child fixture
     - wait for ready output
     - send the wrapper a shutdown signal
     - assert the wrapper exits within a tight timeout
     - assert the exit code is `0`
     - assert the child process is no longer alive after wrapper exit
   - Run the test and confirm it fails for the current implementation if the wrapper is the fault surface.

   - In `frontend/scripts/playwright-web-server.mjs`:
     - make shutdown idempotent

     - ensure wrapper exit does not wait forever on inherited handles once shutdown begins
     - preserve non-zero exit behavior for genuine child startup/runtime failures
